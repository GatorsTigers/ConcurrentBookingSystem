Deploying a mysql on docker:
  1. docker pull mysql
  2. docker run --name <mysql-container> -e MYSQL_ROOT_PASSWORD=<password> -p 3306:3306 -d mysql
  3. docker cp path/to/db_dump.sql <mysql-container>:/db_dump.sql
  4. docker exec -it <mysql-container> mysql -uroot -p
  5. create database <db_name>
  6. use db_name;
  7. source /path/to/mysqldump
